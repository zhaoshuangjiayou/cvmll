研究领域

图像修复

指重建的图像和视频中丢失或损坏的部分的过程。

  • 1.图像补全：图像补全基于深度学习和计算机视觉技术自动修复、补全或者是复原图像中丢失或损坏的区域。
  • 2.图像去雾：图像去雾运用图像处理和机器学习技术对雨天图像进行自动雨滴去除，快速地回复出清晰、高质量的图像处理技术.
  • 3.图像去雨：图像去雨去除图像中的噪声 (即雾)，提高图像的对比度，从而恢复出清晰的无雾图像。
  • 4.图像超分辨：图像超分辨率是指由一幅低分辨率图像或图像序列恢复出高分辨率图像。图像超分辨率技术分为超分辨率复原和超分辨率重建。

图像编辑

图像编辑是利用计算机视觉和深度学习技术对图像进行自动编辑处理的技术，主要应用于虚拟发型、虚拟试衣、一键试/卸妆、风格迁移等。

  • 1.虚拟发型：虚拟发型（Virtual Haircut）是利用生成式对抗网络，对人物图像进行编辑，实现人物发型的一键更换，可广泛应用于个性化美发行业。
  • 2.虚拟试衣：虚拟试衣（Virtual Try-on）是利用图像检测和图像合成技术，将试衣用户的衣服区域替换为服装图像，并无缝融合成新图像的图像编辑技术。是一种让客户方便快捷地体验新衣试穿效果的新应用，可应用于新零售等领域。
  • 3.虚拟试/卸妆：“虚拟试妆”（Virtual Makeup）是结合了GAN、人脸识别技术和图像生成的图像编辑技术，通过捕捉人脸上的关键数据点，对嘴唇特征点等进行精准定位，利用图像处理技术模拟人脸各种化妆技术，让用户方便直观的体验妆容，可应用于个人、线上化妆店、婚纱摄影机构。
  • 4.风格迁移：风格迁移（Style Transfer）是将一幅图像的风格与另一幅图像的内容进行合成的技术，原图片经过一系列的特征变换，具有了新的纹理特征。可以应用在油画风格迁移、景色照片时间迁移和民族服饰风格融合等方面。

目标检测

目标检测的任务是找出图像中所有感兴趣的目标，并确定它们的类别和位置。所以，目标检测任务要解决的问题就是“物体在哪里” 以及“物体是什么”的整个流程问题。

  • 1.显著性目标检测：显著性目标检测（Salient Object Detection）是基于深度学习、计算机视觉技术检测出图像/视频中最显著的目标物体，并将其完整地分割出来。其被广泛用于行人重识别、基于内容的图像搜索、视觉跟踪等领域。
  • 2.伪装目标检测：伪装目标检测（Camouflage Object Detection）是利用人工智能和图像处理等技术将场景中纹理、颜色与背景非常相似的物体检测出来，可应用于产品缺陷检测、物种保护、艺术创作以及医学病灶区域检测等领域。
  • 3.特定目标检测：特定目标检测（Special Object Detection）旨在检测图像/视频中的特定目标，如车辆，行人，交通标志等。在智慧交通等多个领域均极具应用价值，已广泛应用于自动驾驶，智能监控以及公安刑侦等方面。
  • 3.通用目标检测：通用目标检测（Generic Object Detection）目的是识别/定位出自然场景图像中各种类型的目标对象，可应用在许多视觉任务中，如遥感图像处理、无人机检测和机器人应用等。

图像分割

图像分割是指将图像分成若干具有相似性质的区域的过程，主要有基于阈值、基于区域、基于边缘、基于聚类、基于图论和基于深度学习的图像分割方法等。

  • 1.语义分割： 语义分割（Semantic Segmentation）是按目标对象的内容或类别进行分割。在无人车驾驶、医疗影像分析、机器人等领域有广泛的应用前景。
  • 2.实例分割： 实列分割（Instance Segmentation）利用计算机自动从图像中用目标检测方法框出不同实例，再用语义分割方法在不同实例区域内进行逐像素标记。在自动驾驶、地理信息系统、医疗影像分析、工业机器人等方面有着潜在的应用价值。
  • 3.全景分割： 全景分割（Panoptic Segmentation）是语义分割和实例分割的结合。它既可以把所有目标都检测出来，又可以区分出属于同一类别的不同个体。
  • 4.医学分割： 医学图像分割（Medical Image Segmentation）目标是分割出各种医学图像中的病变病灶区域，包括X-ray、显微镜成像、CT、MRI、PET、超声等，可用于影像引导介入诊疗，定向放疗以及手术辅助等。
